# src/shared/tests

Tests for shared types, constants, and utilities.

## Files

- `campaign_types.test.ts`: Tests for campaign progression, leveling, and mission reporting.
- `ItemLibrary.test.ts`: Verifies item definitions and bonuses.
- `PRNG.test.ts`: Ensures deterministic random number generation.
- `types.equipment.test.ts`: Tests for weapon and equipment type definitions.
- `types.test.ts`: Tests for core shared types and their grouped properties.
