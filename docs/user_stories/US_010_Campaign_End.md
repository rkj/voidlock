# US_010: Campaign Conclusion

## Persona & Goal

As a player, I want to see the final results of my campaign effort, whether in glorious victory or crushing defeat.

## Prerequisites

- **Victory**: Boss mission completed.
- **Defeat**: Ironman mission lost OR Bankruptcy reached.

## Action List (Victory)

1. Complete Boss Mission -> Debrief Screen finishes.
1. Observe transition -> Victory Screen opens.
1. Read final summary -> Total Kills, Missions, Roster status.
1. Click "New Campaign" -> Return to Main Menu.

## Action List (Defeat)

1. Lose Ironman Mission -> Defeat Screen opens immediately.
1. Read cause of failure -> "Sector Lost - Ironman Failure".
1. Click "Abandon" -> Return to Main Menu.

## Visual Acceptance Criteria

- Victory Screen has triumphant visuals/icons.
- Defeat Screen has somber/glitchy tactical visuals.
- Both screens summarize "Lifetime Service Record" stats fetched from MetaManager.
