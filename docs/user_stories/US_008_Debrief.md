# US_008: After Action Report

## Persona & Goal

As a Commander, I want to see how my squad performed and who leveled up so that I can plan my next roster changes.

## Prerequisites

- Tactical Mission just completed.

## Action List

1. Wait for mission end -> Debrief Screen opens automatically.
2. Observe Stats -> Total Kills, Time, and Scrap are displayed.
3. Observe Roster -> XP bars animate filling up.
4. Click "Continue" -> Transition back to Sector Map.

## Visual Acceptance Criteria

- Background mission replay is visible behind the UI.
- XP bars show clear segments for "XP Gained".
- "Level Up!" notification appears for units crossing a threshold.
- Scrap tally includes bonuses from Objectives and Elites.
