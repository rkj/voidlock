# src/engine/config

This directory contains configuration files and constants for the game engine.

## Files

- `GameConstants.ts`: Centralized repository for gameplay constants, including hive health, scrap/XP rewards, costs, director parameters, and item stats. Consolidates magic numbers from across the codebase to facilitate balancing and maintenance.
- `CampaignDefaults.ts`: Default values for campaign initialization, such as starting scrap, intel, and unlocked archetypes.
- `DifficultyConfig.ts`: Configuration for different difficulty levels (Simulation, Clone, Iron, Ironman).
